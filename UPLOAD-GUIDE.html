Put Chai Quiz on GitHub

5 easy steps · updates your live site in about a minute

1

Unzip the download

Open the file I gave you (deploy.zip or the folder card) and unzip it. You'll see the game files inside — index.html, helpers.jsx, sb-social.jsx, and the rest.

2

Open your repo → Add file

Go to github.com/aayansh/chai-quiz. Click the Add file button (top-right), then Upload files.

3

Select all files & drag them in

Open the unzipped folder, press Ctrl+A (or ⌘+A on Mac) to select every file inside, then drag them onto the GitHub upload box.

⚠️ Drag the files themselves — not the folder around them.
4

Overwrite & Commit

If GitHub asks to replace the old files, say yes / overwrite. Scroll down and click the green Commit changes button.

5

Wait a minute, then hard-refresh

Give it ~1 minute. Then open your site and do a hard refresh to clear the old cached version:

Ctrl+Shift+R  (Windows/Chromebook)  ·  ⌘+Shift+R (Mac)

https://aayansh.github.io/chai-quiz
✅ Done — your class sees the new version!
